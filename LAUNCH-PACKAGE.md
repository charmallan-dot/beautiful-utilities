## 🛠️ HUMAN SETUP GUIDE:

### STEP 1: Upload Website (30 min)

**What You Need:**
- FileZilla Client (free)
- Profreehost account (free)
- FTP credentials

**Steps:**
1. Open FileZilla
2. Enter FTP credentials
3. Upload all files from `/website/` to `/htdocs/`
4. Test your domain

**Detailed Guide:** `skills/profreehost-deploy/SKILL.md`

---

### STEP 2: Set Up Google AdSense (1-2 hours)

**What You Need:**
- Google account
- Website live with content
- ~10-20 visitors/day (wait a few days after launch)

**Steps:**
1. Go to https://adsense.google.com
2. Sign up with Google account
3. Add your website URL
4. Copy AdSense code
5. Paste code into website header (before `</head>`)
6. Wait for approval (24-48 hours)
7. Ads start showing automatically!

**Pro Tip:** Place ads above the fold but not intrusive
