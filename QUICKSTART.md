### Step 1: Sign Up (2 min)
1. Go to https://profreehost.com
2. Click "Sign Up"
3. Verify your email

### Step 2: Create Website (1 min)
1. Click "Create Website"
2. Choose subdomain: `beautiful-utils.rf.gd` (or similar)
3. Click "Create"

### Step 3: Upload Files (2 min)
1. Click "Manage" → "File Manager"
2. Go to `htdocs/` folder
3. Delete any existing files
4. Upload ALL files from this folder:
   - All `.html` files (11 files)
   - `css/` folder
   - `js/` folder
   - `README.md` (for your reference)

### Step 4: Test! (1 min)
1. Visit your URL: `https://beautiful-utils.rf.gd`
2. Test all 5 tools
3. Check mobile view

## 🎨 Customization

### Change Brand Colors
Edit `css/styles.css`, find `:root`:
```css
--accent-color: #00d9ff;  /* Change this */
```

### Change Background
Edit `css/styles.css`, find `body`:
```css
background: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
```

### Add Your Email
Edit `contact.html`, replace:
```
hello@beautifulutils.com
```

## 🆘 Troubleshooting

**Tools not working?**
- Check that `css/` and `js/` folders uploaded correctly
- Clear browser cache (Ctrl+F5)

**Currency converter offline?**
- Uses free API with fallback rates built-in
- Will work even if API is down

**Contact form not sending?**
- Sign up at https://formspree.io (free)
- Replace form action with your endpoint
